# smartmet-library-giza

Part of [SmartMet Server](https://github.com/fmidev/smartmet-server). See the [SmartMet Server documentation](https://github.com/fmidev/smartmet-server) for an overview of the ecosystem.

## Overview

The giza library provides color mapping and SVG rendering capabilities for SmartMet Server rendering tools. It handles the conversion of data values to colors using configurable palettes and color maps.

## Features

- Color palette management
- Color tree structures for efficient color lookup
- Data-to-color mapping for visualization
- SVG output generation

## Usage

Used by SmartMet rendering tools such as [smartmet-qdcontour](https://github.com/fmidev/smartmet-qdcontour) and [smartmet-qdcontour2](https://github.com/fmidev/smartmet-qdcontour2) to apply color maps to weather data visualizations.
